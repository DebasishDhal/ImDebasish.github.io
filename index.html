Portfolio

  1. Retrieval of Atmospheric Properties using INSAT-3DR Satellite data (Master's Thesis)
    • INSAT-3DR produces radiometric data over the Indian side of the globe with an accuracy of 4 km, while CLOUDSAT produces direct measurements of Cloud properties wih a resolution of 1.3 km, albeit with a poor temporal resolution, since the satellite is constantly moving.
    • Goal - To emulate the readings of CLOUDSAT satellite using the data from INSAT-3DR.
    • The radiometric data from geostationary satellite INSAT-3DR was collocated against cloud-related data from polar satellite CLOUDSAT.
    • The collocated data was fed into ML algorithms to produce cloud-parameters like cloud/clear mask, cloud top height and cloud thickness from INSAT-3DR data.
    • Requirements : - Python, Machine Learning, Process Automation, Data Visualization, Remote server operations, Mapping .

  2. Youtube Playlist Statistics (Web Interface deployed for Public use)
    • For any public/unlisted playlist, it generates the total length and average duration of the playlist's videos.
    • For any two given public/unlisted playlists, it outputs the videos which are present in one but not in the other playlist (symmetric difference). It was the main goal of the pet-project.
    • Requirements : - API Usage, Interface Deployment.

  3. Language Transliteration Project (Web Interface deployed for Public use)
